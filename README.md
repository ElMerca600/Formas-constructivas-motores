
# ⚙️ Formas constructivas de motores eléctricos

## 📌 Descripción
Aplicación para identificar y clasificar las formas constructivas de motores eléctricos de baja tensión.

## 🎯 Objetivo
Facilitar la identificación de configuraciones de montaje de motores en tareas de:
- mantenimiento
- montaje
- reemplazo en campo

## 🧠 Alcance
Esta herramienta permite:
- Identificar formas constructivas de motores
- Comprender las distintas configuraciones de montaje
- Obtener una referencia rápida para uso práctico

## 🧠 Criterios técnicos
- Clasificación basada en estándares industriales
- Identificación de tipos de montaje más comunes
- Aplicación en motores trifásicos de baja tensión

## 🚀 Uso
1. Abrir el archivo `index.html`
2. Seleccionar el tipo de configuración o montaje
3. Obtener la forma constructiva correspondiente

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript
